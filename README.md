\# Programación 1 - UTN



Repositorio con los trabajos prácticos.



\## Trabajos prácticos



TP-Archivos

\*\*Manejo de archivos de texto en Python.\*\*

Programa que crea, lee, agrega y actualiza productos almacenados en un archivo `.txt`, usando listas de diccionarios para manipular la información en memoria.



\- `productos.py`: funciones para crear, leer, agregar, buscar y guardar productos.

\- `main.py`: programa principal con el flujo de ejecución.



TP-Recursividad

\*\*Aplicación de la recursividad en problemas matemáticos y lógicos.\*\*

Conjunto de funciones recursivas (factorial, Fibonacci, potencia, conversión decimal a binario, palíndromos, suma de dígitos, conteo de bloques en pirámide y conteo de dígitos), cada una resuelta con su propio caso base y caso recursivo.



\- `recursividad.py`: todas las funciones recursivas y sus funciones auxiliares de prueba.

\- `main.py`: menú interactivo para probar cada ejercicio por separado.



\## Ejecutar



Desde la carpeta de cada TP, correr:



```bash

python main.py

```
